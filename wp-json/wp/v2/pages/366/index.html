#BOB<\/p>\n\n\n\n

Mining bobcoin is fun and easy!  In just a few steps, you can start earning #BOB by mining blocks on the bobcoin network!<\/h3>\n\n\n\n

You may have heard that mining cryptocurrency is a technical process that involves expensive hardware, except this is not necessarily true during the birth of a new blockchain.  Since the bobcoin network is very new, the mining difficult is currently easy enough to successfully mine on most modern CPUs.  It is very possible to mine thousands of #BOB today while this difficulty remains low.<\/p>\n\n\n\n

This article will cover to configuring the bobcoin wallet and mining script on Windows. If you are interested in running on Linux view the notes in the latest release for detailed instructions.<\/p>\n\n\n\n

Let\u2019s get started!!!<\/h3>\n\n\n\n

1.)<\/strong> Begin by downloading the latest Windows release of bobcoin<\/a>.<\/p>\n\n\n\n

2.)<\/strong> Inside the .zip file you will notice the wallet program, bobcoin-qt.exe and a sample configuration name bobcoin.conf<\/p>\n\n\n\n

Extract the contents of the .zip to a directory of your choice and run bobcoin-qt.exe.<\/p>\n\n\n\n

When bobcoin-qt.exe is first run it will ask you to select a data directory, if you do not change the data directory and use the default settings, it will create a bobcoin directory in %AppData%\/bobcoin.  Take note of the directory as we will use it in step 4.<\/p>\n\n\n\n
